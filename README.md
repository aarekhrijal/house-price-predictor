# House Price Predictor

## Overview
ML model that predicts house prices using XGBoost.

## Results
- Random Forest MAE: $17,693
- XGBoost MAE: $16,415
- Kaggle Score: 0.14577
- Kaggle Rank: 3274

## Tools Used
- Python
- XGBoost
- Scikit-learn
- Pandas
- NumPy
